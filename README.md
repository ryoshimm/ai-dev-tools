# ai-dev-tools
個人用の AI コーディング補助ツール群。

## ai-wt セットアップ
```bash
chmod +x install.sh
./install.sh
```

ai-wtコマンドを実行できるようになる。

- ai-wt create <ブランチ名>
- ai-wt remove <ブランチ名>
- ai-wt list
- ai-wt help

## CLAUDE.mdへの追記
すでに `.claude/CLAUDE.md` が存在する場合は、
`templates/add-claude.md` の内容を確認のうえ、
必要に応じて `.claude/CLAUDE.md` に追記してください。

このファイルは ai-dev-tools 利用時の
最小限の並列AI開発ルールのみを定義しています。
